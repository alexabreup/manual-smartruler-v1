# Visualizador 3D do Smart Ruler

Visualize o modelo 3D do dispositivo Smart Ruler com as instruções de montagem e conexão.

<SmartRuler3DViewer />

## 📋 Sequência de Instalação

O visualizador 3D mostra as 4 etapas de conexão do Smart Ruler:

1. **🔌 Conexão 1** - Alimentação do Dispositivo (topo)
2. **🌐 Conexão 2** - Alimentação do Modem (meio-superior)  
3. **💻 Conexão 3** - Alimentação do PC MUC (meio-inferior)
4. **🔗 Conexão 4** - Cabo USB para PC (base)

## ℹ️ Sobre o Modelo

O Smart Ruler é um dispositivo de distribuição de energia com:
- **4 pontos de conexão** claramente identificados
- **Rotação automática** para visualização completa
- **Setas indicativas** para sequência de montagem
- **Design compacto** para instalação versátil

## 🔧 Características Técnicas

- **Base resistente** em material termoplástico
- **Padrão brasileiro** (2 pinos + terra)
- **Indicadores visuais** para status das tomadas
- **Design compacto** para instalação versátil

Para informações detalhadas sobre instalação e uso, consulte o [Manual de Instalação](/reference/instalacao).
